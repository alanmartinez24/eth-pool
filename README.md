# ETHPool - Smart Contract

### Install dependencies(Node >= 14)
`yarn install`

### Run Tests
`yarn hardhat test` or `npx hardhat test`

### Run task to query the amount of ETH held in smart contract.
`yarn hardhat contract-balance --network ropsten`
